⏱ Cron Jobs
Automated scheduled tasks managed via Ansible
📁
Full cron documentation is version-controlled in Git:
github.com/rsdaroy/helpers.git → ansible/crons/
Managed via php_vars.yml and sync_vars.yml
⚙ Managing Crons
# Install/sync cron jobs $ ./bin/run_crons sync_install # Copy updated scripts from sync_vars.yml to system $ ./bin/run_crons sync_copy # Prerequisite package $ apt install ansible
🔧 Enable / Disable a Cron
# To disable: edit php_vars.yml or sync_vars.yml state: absent # To enable: state: present
🔑 SSH Keys
# Extract ansible SSH keys to: Linux: ~/.ssh/ Windows: C:\Users\<username>\.ssh\
Known Cron Machines Refer to Git for full list
